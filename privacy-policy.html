Third-Party Data Processors

Teacher’s Buddy is a fully serverless application, utilising a variety of cloud-based services for hosting, processing, and storage. We work with third-party providers to enhance functionality while ensuring data security and compliance with privacy laws. Below is an overview of the key third-party data processors involved in our platform.

1. Data Storage & Hosting

Our infrastructure relies on third-party services, which store and process data in the United States. As a result, your personal information is stored, transferred, and accessed outside of New Zealand. We ensure that any overseas disclosure of personal information complies with the New Zealand Privacy Act 2020 and the New Zealand Privacy Principles, including implementing appropriate safeguards to protect your data.

  • Database: Managed MySQL provided by PlanetScale (USA).
  • File Uploads: Files are stored using Amazon S3 via UploadThing (USA).
  • File Embedding & Search: We use ChromaDB (self-hosted on Railway) for document storage and retrieval.

2. AI Processing

To enhance user experience, we integrate AI-powered tools that process user-generated content:

  • AI Framework: We use LangChain to communicate with the OpenAI API.
  • Response Streaming: AI-generated content is streamed using Vercel AI SDK with Edge Runtime.
  • Data Handling: User-generated AI content is automatically saved unless manually deleted by the user.

3. Error Tracking & Feature Flags

  • Error Monitoring: We use Sentry to track system errors and improve performance. Some logs may include anonymised user interaction data for debugging purposes.
  • Feature Flags & Personalisation: Flagsmith helps us manage feature rollouts and personalise user experiences.

4. Authentication & Security

  • User Authentication: We use NextAuth for secure authentication. Depending on the authentication provider (e.g., Google, Microsoft), personal information may be processed according to their respective privacy policies.

5. Deployment & Processing Locations

  • Application Hosting: Our platform runs on Vercel, which uses edge computing to process requests efficiently across global data centres.
  • Data Processing: All personal data is processed in the United States, in line with the New Zealand Privacy Act 2020.
